HyperText Markup Lenguaje

HTML (HyperText Markup Language) es el lenguaje base de la web. Su función principal es estructurar el contenido que se muestra en los navegadores, permitiendo organizar textos, imágenes, enlaces y otros elementos. No es un lenguaje de programación, sino de marcado, ya que utiliza etiquetas para indicar cómo debe presentarse cada parte de la información.

Características principales de HTML:

  • Es un lenguaje de marcado, no de programación.
  • Utiliza etiquetas (tags) delimitadas por <> .
  • Funciona en conjunto con otros lenguajes como CSS (estilos) y JavaScript (interactividad).
  • Evoluciona constantemente, siendo HTML5 la versión más moderna y utilizada hoy en día.

Etiquetas esenciales

  • <html>: define el inicio y fin de un documento HTML.
  • <head>: contiene metadatos, enlaces a hojas de estilo, scripts, y el título de la página.
  • <body>: incluye todo el contenido visible para el usuario.

Contenido básico

  • Títulos (<h1> – <h6> ): permiten jerarquizar el contenido, desde el título principal (<h1>) hasta subtítulos más pequeños (<h6>).
  • Párrafos (<p>): agrupan bloques de texto.
  • Enlaces (<a>): conectan una página con otra mediante la propiedad href.
  • Imágenes (<img>): insertan elementos gráficos en la página.

Organización de contenido

  • Listas no ordenadas (<ul>): muestran ítems con viñetas.
  • Listas ordenadas (<ol>): muestran ítems numerados.
  • Listas de definición (<dl>): útiles para glosarios o pares término–definición.
  • Tablas (<table>): organizan datos en filas y columnas con etiquetas como <tr> (fila), <td> (celda) y <th> (encabezado).

Interacción con formularios

  • <form>: crea formularios interactivos.
  • Elementos de entrada (<input>, <textarea>, <select> ): permiten al usuario escribir datos o seleccionar opciones.
  • Botones (<button>): envían la información o realizan acciones.
