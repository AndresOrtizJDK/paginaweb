Cascading Style Sheets

CSS (Cascading Style Sheets) es el lenguaje utilizado para dar estilo y presentación al contenido estructurado en HTML. Gracias a CSS, una página web pasa de ser un conjunto de texto e imágenes simples a un diseño visual atractivo, organizado y adaptable.

Su función principal es separar el contenido (HTML) del diseño (CSS), lo que permite mantener un desarrollo más ordenado, escalable y fácil de mantener.

Características principales de CSS

  • Define colores, tipografías, tamaños, fondos y márgenes.
  • Permite controlar la disposición de los elementos en la pantalla.
  • Facilita la creación de diseños responsivos que se adaptan a diferentes dispositivos.
  • Se aplica mediante tres formas principales:
    • Inline: dentro de un atributo style.
    • Interno: dentro de la etiqueta <style> en el documento HTML.
    • Externo: en un archivo .css separado, vinculado con <link>.

Temas básicos de CSS

Selectores y propiedadess

Los selectores indican qué elementos de HTML se van a modificar, y las propiedades especifican el estilo que se aplicará.

Ejemplos
  • p { color: blue; } → todos los párrafos en azul.
  • .clase { font-size: 18px; } → afecta a elementos con esa clase.
  • #id { text-align: center; } → centra el texto de un elemento con ID.

Colores y fuentes

CSS permite modificar la apariencia del texto y otros elementos:

  • Colores mediante nombres (red), códigos hexadecimales (#FF0000), o valores RGB/HSL.
  • Tipografías con font-family.
  • Tamaños de texto con font-size.
  • Estilos como negrita (font-weight), cursiva (font-style) y decoraciones (text-decoration).

Modelo de caja (Box Model)

Cada elemento en HTML se representa como una caja rectangular compuesta por:

  • Contenido (texto o imagen).
  • Padding: espacio interno entre el contenido y el borde.
  • Border: contorno que rodea el elemento.
  • Margin: espacio externo que separa un elemento de otro.

Entender este modelo es fundamental para organizar correctamente los diseños.

Posicionamiento de elemento

CSS permite ubicar los elementos en diferentes posiciones:

  • Static: posición por defecto.
  • Relative: desplazamiento relativo a su posición original.
  • Absolute: se posiciona respecto al contenedor más cercano con posición definida.
  • Fixed: permanece en el mismo lugar incluso al hacer scroll.
  • Flexbox y Grid: herramientas modernas para crear diseños responsivos y organizados en columnas y filas.

Responsive Design

El diseño responsivo busca que las páginas se adapten a diferentes tamaños de pantalla (computadoras, tablets, móviles).

  • Se logra principalmente con media queries:

@media ( max-width : 768px ) {
body {
background-color: lightgray; }
}
